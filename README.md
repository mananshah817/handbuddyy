# HandBuddyy - Premium Bag Store

HandBuddyy is a modern, handcrafted e-commerce platform for premium tote and sling bags. Built with a focus on elegance and utility, the site provides a seamless shopping experience.

## ✨ Features

- **Dynamic Product Catalog**: Browse a curated collection of handcrafted bags.
- **Firebase Integration**: Real-time inventory and order management using Google Firebase.
- **User Authentication**: Secure signup and login for customers.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Shopping Cart**: Smooth cart management and checkout flow.
- **Admin Dashboard**: Manage products and orders in real-time.

## 🚀 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Backend**: Firebase Firestore (Database)
- **Deployment**: GitHub Pages (hosting)

## 🛠️ Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mananshah817-commits/handbuddyy.git
   ```

2. **Open the project**:
   Simply open `index.html` in any modern web browser.

3. **Firebase Configuration**:
   The Firebase config is located in `js/firebase-init.js`. For a production environment, ensure your Firebase Security Rules are properly configured.
